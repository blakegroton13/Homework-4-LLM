\# Homework 4 - LLM API Interaction



\## Overview



This project demonstrates how to interact with a Large Language Model (LLM) using code instead of a web interface. The script sends multiple prompts to a Groq API and prints the responses.



\## What the code does



\* Connects to a free LLM API (Groq)

\* Sends different prompts to the model

\* Receives and prints responses

\* Shows how outputs change based on input prompts



\## Prompts Used



1\. Explain artificial intelligence in one sentence

2\. Explain artificial intelligence in detail with examples

3\. Explain artificial intelligence like I am a beginner



\## How to run



1\. Install Python



2\. Install dependencies:



```

python -m pip install -r requirements.txt

```



3\. Add your API key in the script:



```

API\_KEY = "YOUR\_API\_KEY\_HERE"

```



4\. Run the program:



```

python homework4.py

```



\## Notes



This assignment focuses on learning how to use LLMs through code and experimenting with different inputs and outputs.
